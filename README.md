# Автотесты для веб-приложения "Яндекс.Самокат"
Проект разработан с использованием паттерна **Page Object Model (POM)**.

---

Тесты проверяют два основных функциональных блока: FAQ и сценарии оформления заказа.

### 1. Тесты выпадающего списка (FAQ)

Файл: `test_drop_list_questions.py`

| Тест | Описание |
| :--- | :--- |
| `test_drop_list_questions` | Проверяет, что при клике на **каждый из 8 вопросов** в разделе **"Вопросы о важном"** отображается соответствующий **корректный ответ**. Используется параметризация для покрытия всех 8 пар вопрос-ответ. |

### 2. Тесты оформления заказа и навигации

Файл: `test_orders.py`

| Тест | Описание |
| :--- | :--- |
| `test_order_page_script_1` | **Позитивный сценарий заказа** с использованием **верхней** кнопки "Заказать" и выбором **черного** самоката на **сутки**. Проверяется наличие окна **"Заказ оформлен"**. |
| `test_order_page_script_2` | **Позитивный сценарий заказа** с использованием **нижней** кнопки "Заказать" и выбором **серого** самоката на **двое суток**. Проверяется наличие окна **"Заказ оформлен"**. |
| `test_transition_to_main_page_via_logo_scooter` | Проверяет, что клик по **логотипу "Самокат"** переводит пользователя на Главную страницу (`/`). |
| `test_transition_to_dzen_page_via_logo_yandex` | Проверяет, что клик по **логотипу "Яндекс"** открывает в новой вкладке главную страницу **Дзена** и содержит заголовок "Новости". |
